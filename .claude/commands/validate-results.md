Validate sponsor search results for quality and completeness.

For each sponsor in the results, check:

1. **Existence Verification**
   - Organization actually exists
   - Has online presence (website)
   - Legitimate organization

2. **Contact Information Quality**
   - Valid email format
   - Phone number provided
   - Website accessible
   - Physical address available

3. **Giving Program Verification**
   - Has active giving/grant program
   - Recent donations (last 2 years)
   - Grant size matches our needs
   - Application process clear

4. **Mission Alignment**
   - Supports mental health causes
   - Funds arts education
   - Geographic match (NYC/NY)
   - Target demographics align

5. **Quality Score** (1-10 for each sponsor):
   - 9-10: Perfect match, all info verified
   - 7-8: Good match, most info verified
   - 5-6: Possible match, some gaps
   - 1-4: Poor match, remove from list

**Output:**
- Validated sponsor list (removed low-quality)
- Quality report with issues found
- Recommendations for follow-up
- Final count by confidence level
