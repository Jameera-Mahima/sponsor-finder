# Healing NY Sponsor Finder Project

## Mission & Vision
This system finds sponsors for Healing NY - a community school focused on:
- Mental health support
- Arts education
- Community healing

## Project Goal
Build an intelligent sponsor search system that:
1. Takes prompts with requirements (budget, location, sector)
2. Extracts keywords from prompt + mission/vision
3. Performs web crawling and search
4. Returns categorized sponsor lists (corporations, foundations, NGOs, individuals)

## Technical Requirements
- Web search and crawling capabilities
- Keyword extraction from mission/vision
- Sponsor categorization
- Result compilation and formatting
